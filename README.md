# Simple node + express + typescript boilerplate

## Steps to start:
1. Create .env file with two values
```
NODE_ENV=development
SERVER_PORT=8080
```

2. Install packages with yarn or npm  
```yarn```   
or  
```npm```  

3. Run with yarn or npm  
```yarn start```  
or  
```npm start```

## Test with linter
Run this command:  
```yarn test:lint```
